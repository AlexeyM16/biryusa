Штурман Енисея: Цифровой помощник для аэролодки Raptor 650
Offline-first приложение для планирования маршрутов аэролодки Raptor 650. Система является инструментом поддержки принятия решений, анализирующим навигационные условия, расход топлива, временные затраты и риски до выхода на воду.

Ссылка на shortcut:
https://disk.yandex.ru/d/fa4UIUo9g47lIA

Основные возможности
Маршрутизация: Построение оптимального пути между точками старта и назначения с учетом типов покрытия (вода, лед, мелководье, камни, болото, шуга).

Режимы движения: Выбор приоритета прохождения: Быстрый (минимум времени), Экономичный (минимум топлива), Кратчайший (минимум дистанции), Безопасный (минимизация риска).

Расчетная модель: Учет конфигурации лодки (базовая / с поддувом), веса пассажиров, температуры воздуха и текущего запаса топлива.

Контроль безопасности: Автоматическое предупреждение о недостаточном запасе топлива, высоком риске, потере глиссирования или попадании на участки с твердым покрытием при несовместимой конфигурации.

Автономность (Offline-first): Проект спроектирован для работы без постоянного подключения к интернету. Скрипт Service Worker кэширует интерфейс и картографические тайлы на стороне клиента.

Отладка и верификация: Встроенный модуль синтетических тестов для проверки математической модели и генерации файла answers.json для экспертной оценки.

Технические требования
Backend:

Python 3.10+

FastAPI — REST API

Pydantic — валидация входных данных

heapq — алгоритм A* для поиска оптимального пути

Frontend:

Vanilla JS, HTML5, CSS3

Leaflet.js — картографический движок

Canvas API — визуализация графа в режиме отладки

GIS и данные:

Shapely — геопространственный анализ

GeoJSON — формат топологических данных
